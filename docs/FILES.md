# WiFiSwitch

## Configuration files

### Network settings (config.xml)

Parameter | Description | Default
----------|-------------|---------
name      | Device host name | sensor
ntp1      | Name of NTP server 1 | 192.168.1.1
ntp2      | Name of NTP server 2 | 192.168.1.1
ntp3      | Name of NTP server 3 | pool1.ntp.org
timezone  | Local time zone offset | 0

### *Example:*
```xml
<config>
<name>sensor</name>
<ntps1>192.168.0.1</ntps1>
<ntps2>pool1.ntp.org</ntps2>
<ntps3>pool2.ntp.org</ntps3>
<timezone>-4</timezone>
</config>
```
